# PhotoS 포토폴리오
=============
      
간단한 검색 앱이라서
영상이 아닌 .jpg 로 소개하겠습니다
레트로핏 과rx 공부용 프로젝트입니다
      
개발언어 . Kotlin
개발 프로그램 . Android Studio
     
라이브러리 . retrofit2, gson , okhttp , glide , rxjava, rxkotlin, rxbinding,coroutines
        
레트로핏을 활용하여 unsplash api 호출
